# BUSI Breast Ultrasound Classification

This project uses deep learning to classify breast ultrasound images into three classes:

- Benign
- Malignant
- Normal

The BUSI (Breast Ultrasound Images) dataset was used for training.

## Model
The model is based on VGG16 transfer learning.

## Method
1. Dataset preprocessing
2. Train / Validation / Test split
3. Model training using VGG16
4. Evaluation using accuracy, precision, recall and F1-score

## Results
The model achieved around 85% accuracy after training for 30 epochs.
